# Alessia's Pizza & Pasta Website

Static website for Alessia's Pizza & Pasta with menu, specials, and ordering links.

## Run locally
```bash
python3 -m http.server 8000
```
Then open http://localhost:8000.

## Asset notes
- Replace `assets/logo.svg` with the provided logo file.
- Replace `assets/hero.svg` with the grilled chicken penne hero image.
